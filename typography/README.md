## Usage
Recommended for blog posts pages use. CSS files are primarly based on styling HMTL tags such as ```<article>```,  ```<h1>, <h2>``` etc. There are also some components which can be initialized by classes - e.g. ``` .wrapper ```. To ``` .wrapper ``` class is assigneted a certain **max-width** that gives "natural" margin to whole page. The ``` .wrapper ``` should be applied on ``` <div> ``` tag.
